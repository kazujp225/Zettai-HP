## ğŸš€ Getting Started

### Prerequisites
```bash
Node.js 18.0+ 
npm, yarn, pnpm, or bun
```

### Installation

1. **Clone the repository**
   ```bash
   git clone https://github.com/kazujp225/Zettai-HP.git
   cd Zettai-HP
   ```

2. **Install dependencies**
   ```bash
   npm install
   # or
   pnpm install
   ```

3. **Run the development server**
   ```bash
   npm run dev
   # or
   pnpm dev
   ```

4. **Open your browser**
   ```
   http://localhost:3000
   ```
